So in order to use this code you install vitest: https://vitest.dev/guide/

# With npm
npm install -D vitest

# or with yarn
yarn add -D vitest

# or with pnpm
pnpm add -D vitest

# TIP
Vitest requires Vite >=v3.0.0 and Node >=v14
